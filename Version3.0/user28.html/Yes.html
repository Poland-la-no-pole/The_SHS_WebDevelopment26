HTML Tags Demo

<p>: Defines a paragraph of text. Type: Block element

<main>: Represents the main content of the webpage. Type: Block element

<em>: Emphasizes text (usually italic). Example: This text is emphasized Type: Inline element

<ol>: Creates an ordered (numbered) list.
<li>: Represents each item in a list. Type: Both are Block elements

  1. First item
  2. Second item
<div>: A generic container used to group content. Type: Block element

<span>: A generic inline container for styling text. Example: Blue text Type: Inline element

<img>: Displays an image.

Type: Inline element

<a>: Creates a hyperlink. Example: Visit Google Type: Inline element

<table>: Creates a table for organizing data. Type: Block element
